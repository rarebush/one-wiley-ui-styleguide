Wiley's UI Living Style Guide

Introduction

Wiley’s UI Living Style Guide is an accessory to the One Wiley Brand Guidelines. The purpose of this style guide is to help inform the design of digital products such as websites, web apps, and mobile apps. Like the One Wiley Brand Guidelines, this living style guide is meant to be flexible, and it will evolve over time. The intention is that UI designers start using it as much as possible over time, so please check back here when starting a new digital initiative.

Currently the toolkit only includes some fundamental elements of the One Wiley visual language and several footer templates. In the future we aim to increase the scope of the One Wiley UI Guidelines to encompass a wide range of HTML and CSS components that can be easily dropped into future websites and applications.

How to use these guidelines

These guidelines have been designed to ensure we have a consistent, yet flexible brand look and feel across all of Wiley. They’ve been divided into nine key sections that should provide a good understanding of what to do and how to do it.

While we’ve provided many examples, there will always be some exceptions to the rules or questions that remain unanswered. If you find yourself in such a situation, then do your best to come to a sensible conclusion. If there’s something that you’re not 100% sure about, then please feel free to contact a member of the UI Living Style Guide team.
