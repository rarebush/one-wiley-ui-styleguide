One Wiley UI Guidelines

This UI toolkit is an accessory to the One Wiley Brand Guidelines. The purpose of the contents here is to document and communicate the various UI elements that appear across the Wiley digital product portfolio. Once inside the toolkit, the assets are easily discoverabe by element type, project, relation to the wiley brand and device.

The UI guidelines are a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex components and page layouts.

Usage

Start by adding the following files to your page:

<link rel="stylesheet" href="assets/toolkit/styles/toolkit.css">
<script src="assets/toolkit/scripts/toolkit.js"></script>
